# 娜娜奇

```
娜娜奇出生在一個偏僻的小村莊「尼格村」，那裡的人自給自足，也跟外界沒什麼交流。
因此當魔物開始襲擊各地時，尼格村還未曾聽聞，還認為自己活在勇者大人所保衛的和平時代。

她出生的那天，隨著嬰兒啼哭，魔物終是來到這個小村子門前。男人們奮勇的與魔物們搏鬥，女人帶著孩子們躲藏在地窖中，
當鄰居看見娜娜奇母親懷中無辜眨著眼的娜娜奇，一眼灰藍，另一眼閃著妖異金黃，
鄰居發出了淒厲的吶喊「她肯定是被詛咒的孩子！一定是她把魔物招來村子的！」
在村民驅逐她的呼聲中，母親緊緊的抱著她，直到父親的死訊傳回來，母親終究無力的鬆開了手。

她最終在村子倖存者遷移時被拋棄在樹林裡，母親雙眼無神的將她在放在一個樹洞中，
一句話也沒說、只是用灰藍色的雙眸靜靜望了她幾秒。那是從出生時便有記憶的娜娜奇對母親最後的印象。
娜娜奇並沒有這麼死去，她被路過的人口販子撿走，賣給了一名武器匠人。

那是一個孤單的老人，名為「龍」。他養育著娜娜奇，同時也鍛鍊著她成為一名合格的冒險者。
「讓自己決定自己的命運，我能夠給你的只有與他人一樣公平的起點—活著。」
龍說著，為她打造了銳利的剪刀，以及藏在玩偶中的盾牌。
在娜娜奇12歲那年，老人在睡夢中去世了。

她掩埋了老人，在他的墓前輕輕說道：「謝謝你給我的起點—一個家。」
一滴淚落在墳土上，娜娜奇帶走老人的項鍊，項鍊中是一張舊照片，一個壯漢與一名笑容燦爛的女孩—那是龍曾經的女兒。
娜娜奇對著鏡子扯了扯嘴角，練習好幾次，終於擺出了和照片中相同的可愛笑容。
「我會活下去的，爸爸。」說著，娜娜奇離開這個長眠著一代武器匠的家。而她的第一站來到了人們所說的冒險者公會，堅定地踏了進去...
```
